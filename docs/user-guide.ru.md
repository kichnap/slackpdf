# Руководство пользователя SlackPDF

SlackPDF — бесплатный инструмент для работы с PDF-файлами на Windows. Не требует установки .NET и не отправляет данные в интернет.

## Интерфейс

Главное окно состоит из боковой навигационной панели и рабочей области. Переключение между модулями мгновенное — состояние каждого сохраняется до закрытия программы.

## Объединить

Склеивает несколько PDF-файлов в один. Поддерживает выбор диапазонов страниц из каждого файла.

<!-- screenshot: модуль Объединить -->

### Как использовать

1. Нажми **Добавить файлы** или перетащи PDF-файлы прямо в таблицу
2. Измени порядок файлов кнопками **↑ ↓** в панели справа от таблицы:
   - Кликни по строке, чтобы выбрать один файл
   - Ctrl+клик или Shift+клик — выбрать несколько файлов; стрелки переместят всю группу сразу
3. В колонке **Выбор страниц** укажи нужные страницы (оставь пустым — все страницы):
   - `1-5` — страницы с 1 по 5
   - `1,3,7` — конкретные страницы
   - `2-` — со 2-й до конца
   - `1-3,7,10-` — комбинация
4. Выбери параметры закладок и AcroForms
5. Укажи выходной файл и нажми **Выполнить**

### Параметры

| Параметр | Описание |
|---|---|
| Закладки | Объединить закладки всех файлов / удалить / создать по одной на файл |
| AcroForms | Удалить поля форм / объединить / объединить с переименованием полей |
| Оглавление | Добавить страницу-оглавление с именами файлов и номерами страниц |

## Разделить

Разбивает один PDF на несколько частей по разным критериям.

<!-- screenshot: модуль Разделить -->

### Как использовать

1. Выбери входной файл кнопкой **Обзор** или перетащи файл
2. Выбери режим разделения
3. Укажи папку для результатов (если не указана — файлы сохраняются рядом с исходным)
4. Нажми **Выполнить**

### Режимы разделения

| Режим | Описание |
|---|---|
| Каждая страница | Каждая страница → отдельный файл |
| Каждые N страниц | Группы по N страниц |
| По номерам страниц | Разрыв в указанных местах, например `3,7,12` |
| По размеру файла | Части не превышают указанный размер (МБ) |
| По закладкам | Разрыв на каждой закладке выбранного уровня |

## Чередование

Перемежает страницы из двух и более PDF-файлов по принципу: стр. 1 из файла A → стр. 1 из файла B → стр. 2 из файла A → стр. 2 из файла B…

Незаменим при работе с двусторонними документами, отсканированными на одностороннем сканере: нечётные страницы в одном файле, чётные — в другом.

<!-- screenshot: модуль Чередование -->

### Как использовать

1. Добавь файлы кнопкой **Добавить файлы** или перетащи их
2. Порядок файлов в таблице определяет порядок чередования
3. В колонке **Порядок** установи галочку **Обратный порядок** для нужных файлов, если страницы в них идут в обратном направлении (это типично для чётных страниц при одностороннем сканировании)
4. Кнопки **↑ ↓** справа от таблицы меняют порядок файлов (поддерживается мультивыделение)
5. Укажи выходной файл и нажми **Выполнить**

> **Совет для двустороннего сканирования:** отсканируй нечётные страницы в файл A, чётные в обратном порядке — в файл B, добавь оба файла и поставь галочку «Обратный порядок» у файла B.

## Поворот

Поворачивает страницы PDF на 90°, 180° или 270°.

<!-- screenshot: модуль Поворот -->

### Как использовать

1. Выбери входной файл
2. Укажи угол поворота: 90° / 180° / 270°
3. Выбери какие страницы повернуть:
   - **Все страницы**
   - **Чётные** / **Нечётные**
   - **Выбор страниц** — введи диапазон вручную или нажми **Выбрать страницы…**
4. Укажи выходной файл (или включи «Перезаписать оригинал»)
5. Нажми **Выполнить**

### Визуальный выбор страниц

Кнопка **Выбрать страницы…** открывает панель с миниатюрами всех страниц документа.

<!-- screenshot: панель выбора страниц -->

- Кликни по миниатюре — страница выделится синим
- **Выбрать все** / **Очистить** — быстрый выбор
- **Применить** — переносит выбранные страницы в поле ввода и закрывает панель

## Извлечь

Извлекает выбранные страницы из PDF в новый файл.

<!-- screenshot: модуль Извлечь -->

### Как использовать

1. Выбери входной файл
2. В панели миниатюр выбери нужные страницы (клик, Ctrl+клик, Shift+клик)
3. Выбери режим вывода:
   - **Один файл** — все выбранные страницы в один PDF
   - **По одному файлу на страницу** — каждая страница отдельно
4. Укажи выходной файл и нажми **Выполнить**

---

## Вставить

Вставляет один PDF внутрь другого.

<!-- screenshot: модуль Вставить -->

### Как использовать

1. Выбери **основной документ** — тот, внутрь которого будет вставка
2. Выбери **вставляемый документ**
3. Укажи позицию вставки:
   - **После страницы N** — вставить после конкретной страницы
   - **Периодически каждые N страниц** — вставлять вставляемый документ повторно
4. Укажи выходной файл и нажми **Выполнить**

## Визуальный сборщик

Самый гибкий инструмент: позволяет собрать новый PDF из произвольных страниц нескольких документов в любом порядке путём перетаскивания миниатюр.

<!-- screenshot: визуальный сборщик -->

### Как использовать

**Панель документов (слева)**

1. Нажми **+** или перетащи PDF-файлы в левую панель
2. Каждый документ получает цветную метку (A, B, C…)
3. Переключайся между документами кликом по вкладке
4. Наведи курсор на вкладку — появится подсказка с именем файла, числом страниц, размером

**Сборка (справа)**

5. Выдели нужные страницы в левой панели (клик, Ctrl+клик, Shift+клик для диапазона)
6. Перетащи выделенные страницы в правую зону сборки
7. Меняй порядок страниц в зоне сборки перетаскиванием
8. Правой кнопкой по миниатюре в сборке — контекстное меню (удалить, в начало, в конец, дублировать)

**Инструменты**

| Кнопка | Действие |
|---|---|
| Авто-порядок | Отсортировать по метке документа и номеру страницы |
| Очистить всё | Удалить все страницы из сборки |
| Сохранить PDF | Собрать и сохранить итоговый файл |
